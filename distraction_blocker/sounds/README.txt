BELL SOUND PLACEHOLDER
======================

To add a notification bell sound:

1. Download any short bell/chime sound (MP3 format)
2. Rename it to "bell.mp3"
3. Place it in this folder (sounds/)

Recommended free sources:
- https://freesound.org (search for "bell" or "chime")
- https://mixkit.co/free-sound-effects/bell/

Keep the file small (under 100KB) for best performance.
